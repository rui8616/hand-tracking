## 主要结论

把手势 / 姿势识别作为独立 To-C 产品的核心价值，在过去几年里**反复被市场证伪**：

- Osmo（被 Byju's $120M 收购后破产）
- Active Arcade（找不到付费模式，转向硬件 Nex Playground）
- 多个拉伸 / 健身 App（被搜索市场、自然传播弱）

**手势 / 姿势识别更适合作为已有产品的辅助功能**（Apple Watch 心率、AirPods 头部追踪、Vision Pro 眼动追踪），而不是独立 App 的核心价值。

## 留这些文档的价值

- 展示「从 demo 出发认真思考商业化」的过程
- 展示「能识别失败模式并诚实退出」的判断力
- 作为后人想做类似方向的**避坑参考**

如果你正在考虑做手势 / 姿势识别的 To-C 产品，强烈建议读一遍 [product-research.md § 7.7](./product-research.md) 的反思部分。
